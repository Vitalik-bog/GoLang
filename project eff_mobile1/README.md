# Subscriptions Service

Микросервис для управления онлайн-подписками пользователей.

## Функциональность

- CRUD операции над подписками
- Фильтрация подписок по пользователю и сервису
- Расчет общей стоимости подписок за период
- Soft delete подписок
- Swagger документация

## Требования

- Go 1.21+
- Docker и Docker Compose
- PostgreSQL 15+

## Установка и запуск

### Локальный запуск

1. Клонируйте репозиторий:

```bash
git clone <repository-url>
cd effective_mobile_service
```
